# asmuelle.github.io

My personal landing page — a summary of selected projects, served at
**https://asmuelle.github.io/**.

- `index.html` — self-contained (no build step, no dependencies). Hand-picked
  *Featured* projects plus a live grid of my non-fork repositories fetched from
  the GitHub API at load time (forks hidden, sorted by stars then recency).

To change what's featured, edit the `Featured` section and the `FEATURED`
array in the inline script.
