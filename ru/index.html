Услуги

Чем я помогаю

Ядра на Rust для нативных приложений

Я переношу критичную по производительности и безопасности логику в безопасное по памяти ядро на Rust, связанное со Swift через UniFFI и общее для macOS и iPadOS.

Инструменты PostgreSQL и SSH

Я проектирую и создаю инструменты для баз данных и удалённого доступа — транзакционно-безопасный DDL, ER-диаграммы, SFTP и мониторинг — нативные, быстрые и надёжные.

CI для Rust и инструменты разработчика

Я создаю инструменты cargo и CI, которые делают разработку на Rust безопаснее: контроль влияния зависимостей, оценка риска диффов и обнаружение расхождений со спецификацией.

Частые вопросы

Доступны ли вы для консалтинга?

Да — я берусь за отдельные проекты по разработке на Rust и нативных приложений. Быстрее всего связаться со мной по электронной почте.

Что вы создаёте?

Нативные приложения для macOS и iPadOS на безопасных по памяти ядрах Rust, продвинутые инструменты для PostgreSQL и SSH и инструменты cargo/CI.

С какими языками и платформами вы работаете?

В основном Rust, Swift, TypeScript и Java — для macOS, iPadOS и кроссплатформенного десктопа через Tauri.

Где вы находитесь?

У Боденского озера на юге Германии; работаю с командами удалённо.
